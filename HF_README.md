# 🛕 Temples of Tamil Nadu Recognition App

This is an AI-powered application that identifies 15 famous temples in Tamil Nadu using CLIP and a custom MLP head.

## Features
- **CLIP Zero-Shot**: Versions 1-4.
- **Trained MLP**: Version 5 (High Accuracy).
- **History & Visit Info**: Complete metadata for tourists.
- **Google Maps Integration**: Navigation links for every temple.

## Deployment Info
This space runs using a custom **Docker** container to ensure all computer vision libraries (OpenCV) work correctly.

**SDK**: Docker / Streamlit
**Port**: 7860
